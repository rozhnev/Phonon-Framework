Phonon

Responsive front-end framework with a focus on simplicity and flexibility

Phonon is an open source HTML, CSS and JavaScript agnostic library that allows to create a website or a hybrid Web app.

Getting started

About

Originally, Phonon has been created for building hybrid apps with Apache Cordova by using web technologies such as HTML, CSS and JavaScript. Of course, there are many frameworks well known, but Phonon gives an intuitive and amuzing way of building apps. For example, the app's workflow can be defined with page events. When the user access on the page-X for the first time, the onCreate callback is called, when the page becomes visible, the onReady callback is called, etc. You can also prevent the automatic close if an action is required.

Easy to use and setup

Rich Docs & JavaScript or HTML based

The documentation is filled with examples and shows the visual result of the components.

JavaScript approach: Many components such as modals and notifications don't require any HTML markup. Use the JavaScript API directly to display components.

HTML approach: Most component options are read directly from the markup. This eliminates the need to define the component markup and setup its JavaScript configuration.

User interface framework

Frontend components

The UI components are designed to support use cases on different device formats such as desktop, tablets and smartphones.

Light weight framework

CommonJS components

Include the components used in your project in ES6+ in order to reduce the size or import the minified files. For more information, please read Custom builds.

Library agnostic

No dependency

Phonon does not use and does not depend on any JavaScript third party libraries.

Choose the right framework according to your needs and your project's size such as React, Angular, Vue.js, jQuery, Zepto.js, Riot.js or work with pure JavaScript.

Site framework

From prototyping to production

Use Phonon as a prototyping website and apps toolkit to show your drafts and mockups or build amazing websites.

Hybrid app framework

Painless SPA

Also known as single page applications, Phonon SPA (phonon-spa.js) comes with simple modules such as Pager and I18n.

  • Pager: is a lightweight page controller and client-side router to organize navigation between pages, read hash parameters and listen to page events such as show, shown, hide, hidden and hash. It also allows you to define page templates and block navigation to a page.
  • I18n: contains texts in different languages and display the text in the active language in views.

Download

The installation can take place in several ways:

  • Download the latest release and use dist files.
  • Clone the repo: git clone https://github.com/quark-dev/Phonon-Framework.git
  • Install with npm: npm install phonon.
  • Install with yarn: yarn add phonon.
  • Install with Bower: bower install phonon.
  • Use CDNJS.
